Proyecto académico

Es una aplicacion web para realizar denuncias sobre la infraestructura pública de Popayán, como deterioro de calles, parques, puentes, edificios, etc. La aplicación web en el lado del cliente es una página SPA dinámica que se conecta a una API hecha en sprng-boot conectada a una base de datos relacional en MySQL.
